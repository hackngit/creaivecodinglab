Creative expression through programming

This companion website forms part of Graham Newman's MRes RCA Communication Design pathway research.

p5.js is a simple, friendy environment to help you learn javascript to make creative, interesting applications happen in the web browser. You might make art, you might make data visualisations you might make websites. There's lots of paths.

p5.js is a library of functions (a walled garden of functions to have fun with - called a sandbox-) to learn some basic functions of drawing using programming. p5.js is also an editor, an application on your computer that you can launch and just start to learn and experiemt with code without having to worry about all the other pieces that you need in order to make something happen in order to build a webpage.
